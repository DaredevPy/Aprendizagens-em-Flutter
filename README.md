# 📚 Aprendizagens em Flutter & Dart

Repositório pessoal com exemplos práticos, técnicas e conceitos fundamentais da linguagem **Dart** e do framework **Flutter**, organizados para estudo e consulta.

---

## 🎯 Objetivo

Este repositório serve como um registro de aprendizado: cada arquivo contém um código com uma técnica,
conceito ou funcionalidade específica, testada e documentada. A ideia é ter uma referência prática 
para consultar sempre que necessário, além de acompanhar a evolução dos estudos.

---

## 📖 O que você vai encontrar

✅ Códigos comentados em português, explicando cada parte  
✅ Conceitos desde o nível básico até o avançado  
✅ Exemplos funcionais e testáveis  
✅ Técnicas recomendadas e boas práticas da linguagem

---

## 📝 Observações

- Este repositório está em constante atualização conforme novos assuntos são estudados.
- Os códigos são feitos com fins de aprendizado — podem ser adaptados e usados livremente.
- Qualquer sugestão ou correção é bem-vinda!
